# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, assume good faith, and keep discussion focused on the technical matter at hand.

## Enforcement

Report unacceptable behavior via GitHub's private vulnerability reporting channel (see [SECURITY.md](SECURITY.md)) or by emailing the repository owner listed in the GitHub profile. All reports are handled confidentially.

Consequences range from a private warning to a permanent ban, depending on severity and pattern.
